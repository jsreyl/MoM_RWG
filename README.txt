This codes calculate the induced current on a p.e.c. scatterer using RWG's approach.
Currently implemented geometries are flat and bent plaques.

To compile simply type
make
on your terminal
and
./a.out
to run.

You need python3's matplotlib, numpy and pandas in order for the figures to be drawn.
For linear algebra calculations used in the cpp codes a version of Eigen 3.3.7 is included, no need to install.
